#  Dvago Website Clone

A modern and responsive **Dvago pharmacy website clone** built using **HTML & CSS**.  
This project replicates the layout and design of an online pharmacy platform, focusing on clean UI and user-friendly experience.

---

##  Features

-  Clean and responsive **Homepage Design**  
-  **Pharmacy Product Sections** (Medicines, Health Products, etc.)  
-  Simple **Navigation Bar** for easy browsing  
-  **E-commerce style layout**  
-  Organized product display  
-  Fully **Responsive Design** (Mobile + Desktop)

---

##  Technologies Used

- **HTML5**  
- **CSS3**  
- Responsive Design (Media Queries)

---


---

## 🚀 How to Run This Project

1. Download or clone this repository  
2. Open the project folder  
3. Double-click on `index.html`  
4. View the website in your browser
